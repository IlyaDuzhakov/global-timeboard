# 🌍 Global Timeboard

**Global Timeboard** — это интерактивное PWA-приложение для отображения точного времени в столицах мира с визуализацией на карте.

Приложение позволяет:
- выбирать города через селектор,
- отображать до 10 аналоговых часов одновременно,
- видеть геометки выбранных стран на карте,
- работать офлайн как Progressive Web App (PWA).

---

## ✨ Основные возможности

- 🕒 **Точное время по IANA-таймзонам**  
  Используются реальные IANA-зоны (Europe/Paris, Asia/Tokyo и т.д.), с учётом DST.

- 🌍 **Интерактивная карта мира**  
  - Цвет стран зависит от площади  
  - Антарктида выделена отдельно  
  - Геометки показывают выбранные страны  
  - Поддержка кликов и tooltip с информацией

- 🧭 **Селектор городов**
  - Поиск по столицам
  - Проверка на дубликаты
  - Ограничение до 10 часов

- 🧱 **Стеклянный UI (Glassmorphism)**
  - Прозрачные контейнеры
  - Мягкие тени
  - Единый визуальный стиль

- 💬 **Единое модальное окно**
  - Используется вместо `alert`
  - Одинаковое поведение во всём приложении

- 📦 **Сохранение состояния**
  - Выбранные города сохраняются в `localStorage`
  - Восстановление после перезагрузки

- 📱 **PWA**
  - `manifest.json`
  - `serviceWorker`
  - Иконки и фавиконки
  - Возможность установки на устройство
  - Работа офлайн

  ## Адаптивность

Приложение адаптировано для работы на настольных компьютерах и мобильных устройствах.  
Интерфейс корректно масштабируется под разные размеры экрана, обеспечивая удобное взаимодействие как на десктопе, так и на смартфонах.

---

## 🛠 Используемые технологии

- **React**
- **Luxon** — работа с датами и IANA-таймзонами
- **D3-geo** — географическая проекция карты
- **TopoJSON**
- **react-select**
- **CSS Modules**
- **PWA (Service Worker, Manifest)**
